# Functionality
- Adding/removing to/from cart
- Separate page for full game info and trailer
- Search by available games
- Simulation of creating an order

# Used technologies
-  TypeScript
-  React
-  Redux Toolkit
-  React Router Dom
-  AntD/AntD Icons
-  axios
